ロールカラーを
自動・自由に。

サーバーのロールカラーを自動でランダム変更。カラーピッカーで好きな色に固定することも。

Discordに追加 コマンドを見る

機能

シンプルな設定で導入できます。

自動カラー変更

指定したロールの色を設定した間隔でランダムに自動変更。60秒〜86400秒で自由に設定できます。

カラーピッカー

ブラウザのネイティブカラーピッカーで色を視覚的に選択してロールに即時反映。

マルチサーバー対応

複数のサーバーに導入しても設定はサーバーごとに独立して管理されます。

権限管理

オーナー承認フローと許可ユーザー制限で、意図しないカラー変更を防止します。

コマンド

スラッシュコマンドで簡単操作。

/change_color 管理者

指定したロールの色を N 秒ごとにランダム変更します。interval_seconds を 0 にすると停止。

role interval_seconds 0 = 停止 / 最小 60 / 最大 86400
/yuu_color 管理者 許可ユーザー

ロールカラーを手動設定します。管理者は対象ロールと許可ユーザーを設定できます。引数なしでカラーピッカーを起動。

role user r g b
